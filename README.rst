===============
nanarino jigsaw
===============

一个传统拼图游戏的demo。

刚接触js的时候的作品。现在用 ``SolidJS`` 浅浅的重构一下。

.. image:: ./jigsaw/logo.svg
    :width: 96 px
    :alt: SolidJS Logo

原 ``jigsaw`` 目录保留作为新的静态资源目录


Clone
=====
::

    # 安装依赖
    pnpm i
    # 运行开发服务器
    npx vite
