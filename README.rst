===============
nanarino jigsaw
===============

一個 `nanarinostyl <https://nanarino.github.io/nanarinostyl/>`_ 主題的 拼圖游戲 demo 。

剛接觸 WEB 时嘅作品。而家用 ``SolidJS`` 淺淺地重構一下。

.. image:: ./jigsaw/logo.svg
    :width: 96 px
    :alt: SolidJS Logo
    :target: https://www.solidjs.com/

原 ``jigsaw`` 目錄保留作爲 vite 的靜態資源目錄

.. highlight:: bash


克隆
=======
::

    # 安裝依賴
    pnpm i
    # 運行開發伺服器
    npx vite


構建
======
::

    npx vite build
